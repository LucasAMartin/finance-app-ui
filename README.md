# Finance App UI

A small Expo React Native UI for tracking finances.

## Prerequisites

- Node.js (16+ recommended)
- npm or Yarn
- Optional: Expo CLI (`npm install -g expo-cli`) or use `npx expo`/`yarn expo`

## Install

1. Install dependencies:

```bash
npm install
# or
yarn
```

## Run (development)

- Start the Metro/Expo dev server:

```bash
npm start
# or
yarn start
```

- Open on Android:

```bash
npm run android
# or
yarn android
```

- Open on iOS (macOS only):

```bash
npm run ios
# or
yarn ios
```

- Run in web browser:

```bash
npm run web
# or
yarn web
```

These scripts are defined in `package.json` and use Expo (see `start`, `android`, `ios`, `web`).

## Notes

- This project is an Expo app using TypeScript. If you don't have the Expo CLI installed globally, use `npx expo start` instead of `npm start`.
- If you see native build errors when running on a device/emulator, ensure your development environment is set up for React Native (Android Studio, Xcode, etc.).
